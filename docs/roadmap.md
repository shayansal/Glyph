# Roadmap

- Phase 0: Prototype.
- Phase 1: Reference engine.
- Phase 2: Mobile host adapters.
- Phase 3: XR host adapters.
- Phase 4: Conformance suite.
- Phase 5: Extension registry.
- Phase 6: Governance foundation.
- Phase 7: Standards proposal.

Future adapters include Web, desktop, iOS/macOS, Android, OpenXR, WebXR, game engines, and server-side semantic snapshot export.
